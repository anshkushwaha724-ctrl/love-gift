# 💕 Aryan & MinMin - Love Gift 💕

A beautiful, interactive romantic web experience created with love! 🌹

## ✨ Features

- **Welcome Screen** - Personalized greeting
- **Countdown Timer** - Days since your first date (September 26, 2024)
- **Memory Game** - Match pairs of hearts
- **Escape Button Game** - Playful interaction
- **Catch Hearts Game** - Click falling hearts for points
- **Scratch Card** - Reveal hidden messages
- **Love Letter** - Heartfelt message from Aryan to MinMin
- **Fireworks Celebration** - Confetti animation
- **Beautiful Design** - Blossom pink theme throughout

## 🎨 Customization

Edit `index.html` to change names, dates, and messages.
Edit `style.css` to change colors (look for `:root` variables).

## 🚀 How to Deploy on GitHub Pages

1. Go to your repository settings
2. Scroll to "Pages" section
3. Select "main" branch as source
4. Your site will be live at: `https://yourusername.github.io/love-gift`
